Introdução ao Sistema

O Sistema de Controle Patrimonial CEAD é uma solução completa para gerenciamento de bens patrimoniais, desenvolvido para atender às necessidades específicas do centro educacional.

Seguro

Sistema com autenticação e controle de acesso

Rápido

Interface otimizada e processamento eficiente

Inteligente

Relatórios e estatísticas em tempo real

Primeiro Acesso e Login

Configuração Inicial

1 Acesse o sistema pela primeira vez

URL: http://localhost/sistema_patrimonial/

2 Crie o usuário administrador

Acesse: http://localhost/sistema_patrimonial/criar_admin.php

Execute este passo apenas uma vez para criar o primeiro usuário.
3 Faça login no sistema

Use as credenciais criadas no passo anterior.

Cadastro de Bens Patrimoniais

Cadastro Individual

1 Acesse a gestão de bens

No menu principal, clique em "Gestão" ou acesse diretamente: sistema_crud.php

2 Clique em "Cadastrar Bem"

Selecione a opção no menu de navegação.

3 Preencha os campos obrigatórios
Campo Obrigatório Descrição Exemplo
Número do Patrimônio Sim Identificador único do bem 5500/2023
Descrição Opcional Descrição do item Computador Dell i5
Localidade Opcional Local onde o bem se encontra Sala 101 - Bloco A
Status Automático Situação do bem Localizado / Pendente
Imagem Opcional Foto do bem (até 5MB) JPG, PNG, GIF

Upload de Imagens

Formatos suportados: JPG, PNG, GIF
Tamanho máximo: 5MB
Redimensionamento automático: As imagens são redimensionadas para 1200x1200 pixels

Importação em Lote via CSV

Estrutura do Arquivo CSV

O sistema aceita arquivos CSV com a seguinte estrutura:

Coluna Campo Obrigatório Descrição Exemplo
A unidade_gestora Não Unidade gestora do bem CEAD
B localidade Não Local onde o bem se encontra Sala 101
C responsavel_localidade Não Responsável pelo bem João Silva
D numero_patrimonio SIM Número único de patrimônio 5500/2023
E descricao Não Descrição do bem Computador Dell

Processo de Importação

1 Acesse a importação

Na página inicial, clique em "Importar Dados" ou acesse: importar.php

2 Escolha a ação para duplicados
Ignorar Duplicados

Mantém os registros existentes e ignora os duplicados do CSV

Sobrescrever Duplicados

Atualiza os registros existentes com os dados do CSV

3 Faça o upload do arquivo

Arraste o arquivo CSV ou clique para selecionar (máximo 50MB).

Importante: O sistema detecta automaticamente a codificação e o delimitador do CSV (, ; ou tab).

Backup e Limpeza

Sistema de Backup

1 Acesse o sistema de backup

No CRUD, clique em "Backup/Limpeza" ou acesse: backup_limpeza.php

2 Clique em "Fazer Backup Agora"

O sistema criará um arquivo SQL completo.

3 Download do backup

Faça download do arquivo para manter em local seguro.

Local dos backups: sistema_patrimonial/backups/
Formato: backup_sistema_patrimonial_AAAA-MM-DD_HH-MM-SS.sql

Limpeza do Banco de Dados

ATENÇÃO: Esta ação é IRREVERSÍVEL! Sempre faça backup antes.
1 Confirmação de backup

Marque a confirmação de que fez backup.

2 Digite "LIMPAR DADOS"

Confirmação por texto para evitar acidentes.

3 Confirmação final

Confirme novamente no popup de alerta.

Segurança: A tabela de usuários NÃO é apagada para manter o acesso ao sistema.

Solução de Problemas

Problemas Comuns

Solução:
  1. Verifique se o usuário foi criado em criar_admin.php
  2. Confirme que o e-mail está correto
  3. Verifique a senha digitada
  4. Execute criar_admin.php novamente se necessário

Solução:
  1. Verifique se o arquivo é um CSV válido
  2. Use delimitadores padrão: vírgula (,) ponto e vírgula (;) ou tab
  3. Abra o arquivo em um editor de texto para verificar a formatação
  4. Certifique-se de que há dados nas colunas

Solução:
  1. Verifique o formato (JPG, PNG, GIF)
  2. Confirme que o tamanho é menor que 5MB
  3. Verifique permissões da pasta uploads/imagens/
  4. Tente redimensionar a imagem antes do upload

Solução:
  1. Verifique se o arquivo existe no diretório correto
  2. Confirme a URL digitada
  3. Verifique as permissões do Apache
  4. Execute o diagnóstico em verificar_sistema.php

Suporte Técnico
